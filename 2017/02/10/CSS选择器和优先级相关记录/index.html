CSS选择器和优先级相关记录

发表于 2017-02-10 | 分类于 CSS

记录CSS的选择器和优先级相关内容，补全遗漏知识点

CSS选择器种类

（1）标签选择器（基本）
常见的body，div。

（2）类选择器（基本）
常见的class=”main”。

（3）id选择器（基本）
常见的#id=”main”。

（4）全局选择器（一般是规定一些基本规则）
用*选择。

（5）组合选择器
一般只针对类和类组合，标签和类的组合。

（6）分组选择器

（7）后代选择器
不一定要是直接的子元素。

（8）子元素选择器

（9）兄弟元素选择器

（10）伪类选择器
IE6不支持。

（11）属性选择器

（12）伪元素（伪对象）选择器

CSS最基本的优先级

1.在属性后面使用 !important的选择器优先级是最高的。

2.CSS的内联样式（作为style属性写在元素内的样式）优先级总是大于外联样式（外部文件或者写在<head>里的<style>标签里的）的。

3.在相同权值的情况下，离被设置元素越近优先级越高，即后出现的样式总是会覆盖前出现的样式。不同权值的情况下则比较权值。

4.创作者的规则高于浏览者，即自定义的样式要优于浏览器设置的样式。

5.继承的样式不如后来指定的样式（子标签会继承父元素的样式）。

CSS选择器优先级

最基本的各级别优先级：
ID选择器 > 类选择器 > 标签选择器、伪类选择器、属性选择器 > 伪元素选择器 > 全局选择器 > 继承得来的 > 浏览器设置。
其他选择器大都是组合的选择器，需要进行权值计算再做比较。

CSS选择器权值计算方法

参考资料：

选择器的特殊性分成4个成分等级：a,b,c,d
等级排列为：a > b > c > d，一个选择器的权值可以表示为a-b-c-d。

（1）a：如果是行内样式或者important，a=1。

（2）b：b表示id选择器的数量。

（3）c：c表示类、伪类、属性选择器的数量。

（4）d：d表示标签、伪元素选择器的数量。

（5）全局选择器的权重可以认为a,b,c,d都为0，基本可以忽略不计。

计算方式：
将a,b,c,d组成的一串数字进行比较，与四位数比较大小从左到右顺序相同。
但0-0-1-0比0-0-0-13的优先级高。

可以说每个等级的差距是十分远的，但是据说可以有256个class类名选择器干掉一个id选择器，因为所有的类名(classes)都是以8字节字符串存储的。8字节所能hold的最大值就是255,当同时出现256个class, 势必会越过其边缘，溢出到id区域。
